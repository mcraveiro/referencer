Referencer

A document organiser and bibliography manager for Gnome

Frequently Asked Questions

Why am I only getting the surname of the first author when I look up metadata?

This issue appears when resolving DOIs using crossref.org. Unfortunately, crossref.org's public OpenURL interface does not provide enough information to fully fill out author information. crossref.org's more fully-featured interfaces are unfortunately not available without subscription. This issue does not affect looking up metadata for ArXiV papers, and any other databases supported in the future.

This issue will be resolved either if crossref.org provides a more full-featured public interface, or if Referencer supports other metadata databases (eg. Medline) in the future.

Is there a Windows or Mac version?

Referencer is designed to be quite portable, using the gtkmm toolkit which can run be used both Windows and Mac platforms. At present Referencer hasn't been compiled on either of these platforms. Any developers wishing to work on supporting non-UNIX platforms are welcome to do so.

What about searching databases such as medline from within referencer?

This is recognised as desirable functionality and may be implemented at some point in the future.

Why are special characters like ß getting mangled in my BibTeX imports?

Referencer assumes that BibTeX files are in ISO-8859-1 (latin1) encoding. Other encodings, such as UTF-8 may cause problems. The reason for this is that vanilla LaTeX doesn't support UTF-8, and Referencer assumes your BibTeX file is compatible with an unmodified LaTeX. A quick workaround (since version 1.0.4) is to open your BibTeX file in a text editor such as gedit, and copy and paste it into Referencer -- this should work if your text editor understands the encoding of your BibTeX file and works properly with the clipboard.

Why isn't Referencer extracting ArXiV IDs from some of my PDFs?

Sometimes the PDFs from arxiv.org aren't formed in such a way that the ID code is readable as plain text. This may be related to whether the paper was produced using LaTeX or Word.
