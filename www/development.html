Development

Mailing list

Please send questions and suggestions to the mailing list.

Subversion

To check out the source code:
svn co svn://svn.icculus.org/referencer/trunk referencer

Web view of the subversion repository:http://svn.icculus.org/referencer/

Translation

Referencer uses intltool for internationalisation. Translation to a new language is quite straightforward, and does not require any programming knowledge. If you'd like to get started, please send a message to the mailing list.
