Referencer

A document organiser and bibliography manager for Gnome

Referencer is a Gnome application to organise documents or references, and ultimately generate a BibTeX bibliography file. Referencer includes a number of features to make this process easier:

  • Smart web links
    Referencer uses documents' DOI codes and Url metadata to provide handy links to the document's web location — no need to maintain your own bookmarks.
  • Import from BibTeX, Reference Manager and EndNote
    No need to start from scratch — Referencer will import your existing bibliography files using the BibUtils library.
  • Tagging
    No need to organise your documents into rigid directory trees — with Referencer you can use tags to categorise your documents.
  • Automatic arXiv metadata retrieval
    If you show Referencer a PDF which has an arXiv ID, Referencer will retrieve the metadata for this document over the internet.

CrossRef users: The crossref.org OpenURL interface now requires user registration, and thus is not working with Referencer at the moment. Alternatives are under investigation for the next release.

Getting Referencer

Note: distribution packages may not always correspond to the latest version of referencer.

Ubuntu package: Get it from GetDeb (7.10 Gutsy users can get Referencer 1.0.4 from apt)

Fedora package: Referencer is available from the fedora-extras repository for Fedora Core versions 6 and 7.

Gentoo Ebuild: Bugzilla thread

Mandriva package:
Mandriva Linux 2007 Spring has Referencer 1.0.2 in its /contrib/release repository, and Mandriva Linux 2007 and 2007 Spring have Referencer 1.0.4 in their /contrib/backports repositories. More information is available on installing software and using backports.

Debian package: Referencer is available in debian unstable

ZenWalk package: Referencer is available in the main ZenWalk repository

Source code: referencer-1.0.4.tar.gz (ChangeLog)

If there is not yet a package for your distribution, you'll need to compile referencer from source. See the development page for more details.
